# nyctailblazers-modern — Project Overview
**Owner:** Kaoz625 | **Stack:** React, Vite, TypeScript, Tailwind CSS | **Status:** Built but incomplete — missing booking, contact form, and social links

## What This Is
A cinematic multi-service portfolio website for NYC Tailblazers, a professional dog walking and pet care business based in New York City. The site showcases services, builds trust with potential clients, and drives bookings. It is intended to be the primary marketing and conversion website for the business.

## Current State
React/Vite/Tailwind app exists with basic structure and images. Missing: booking integration, real pricing, working contact form, social media links, and Cloudflare Pages deployment config.
